2. Tipologia di Applicazione

Financial Authority Database (disponibile come sito web e Progressive Web App) è un database educativo di pubblico dominio che fornisce:

  • Link alle autorità finanziarie di 108 paesi
  • Informazioni pubbliche di contatto delle autorità
  • Strumenti per la protezione dei consumatori contro le frodi finanziarie

3. Dati Personali NON Raccolti

L'applicazione NON richiede né raccoglie:

  • ❌ Nome e cognome
  • ❌ Indirizzo email
  • ❌ Numero di telefono
  • ❌ Indirizzo fisico
  • ❌ Dati di geolocalizzazione
  • ❌ Informazioni finanziarie o bancarie
  • ❌ Dati biometrici
  • ❌ Cronologia di navigazione personale
  • ❌ Cookie di tracciamento
  • ❌ Credenziali di accesso (l'app non richiede registrazione)

4. Funzionamento dell'Applicazione

L'applicazione funziona in modalità completamente offline dopo il primo caricamento:

  • Tutti i dati sono contenuti nel file JSON statico caricato una sola volta
  • Nessuna comunicazione con server esterni per il tracciamento utenti
  • Nessun database backend che memorizza attività degli utenti
  • Service Worker utilizzato SOLO per il funzionamento offline (cache locale)

5. Tecnologie Utilizzate

Service Worker:

  • Utilizzato esclusivamente per memorizzare i file dell'app nella cache locale del dispositivo
  • Permette il funzionamento offline dell'applicazione
  • NON invia alcun dato a server esterni
  • Tutti i dati rimangono sul dispositivo dell'utente

Local Storage:

  • Utilizzato solo per salvare preferenze di lingua (opzionale)
  • I dati rimangono esclusivamente sul dispositivo dell'utente
  • Nessuna sincronizzazione con server esterni

6. Link Esterni

L'applicazione contiene link diretti ai siti web ufficiali delle autorità finanziarie di 108 paesi e organizzazioni internazionali (ESMA, FBI IC3, IOSCO, ScamShield, ecc.).

⚠️ Importante: Quando l'utente clicca su questi link, viene reindirizzato ai siti web esterni delle rispettive autorità. Le politiche sulla privacy di tali siti sono indipendenti e non sotto il controllo di questa applicazione. Consigliamo di leggere le privacy policy di ciascun sito visitato.

7. Hosting e Sicurezza

L'applicazione è ospitata su GitHub Pages:

  • GitHub può raccogliere dati aggregati di traffico (non personalizzati)
  • GitHub Privacy Policy: https://docs.github.com/en/site-policy/privacy-policies
  • Connessione sicura HTTPS obbligatoria

8. Cookie

✅ Questa applicazione NON utilizza cookie di tracciamento o profilazione.

L'unico utilizzo di cookie potrebbe derivare da GitHub Pages per funzioni tecniche essenziali, ma questi NON sono utilizzati per tracciare o profilare gli utenti.
